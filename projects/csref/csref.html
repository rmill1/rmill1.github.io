Introduction

This is a reference page that you can use throughout 7th Grade Computer Science this quarter. It has a lot of information, but probably not everything you might find yourself wondering about. Use this reference as a starting point, and other online resources as you see fit!

  • Open and Close Tags

    Most HTML tags have an open and a close tag. Tags are indicated by angle brackets and the name of the tag is in between the brackets. A close tag has a slash that tells the computer when the tag ends.

    Here are common tags that you will use throught the quarter - note that they include both an opening tag and a closing tag:

    <!-- Example tags --> <h1>This is the largest header</h1> <h6>This is the smallest header</h6> <p>This is a paragraph tag</p>
  • Self-Closing Tags

    Some HTML tags are special and they do not have a close tag with a slash. Here are a few self-closing tags that you will probably use at some point:

    <!-- Example self-closing tags --> <hr> <br> <img src="mypic.jpg">

CSS Styling

There are three types of styling that we will be doing this quarter, starting with your second mini project. Each method of styling share similarities, but the location of the styling changes.

  • Inline Styling - Careers Page
    <h1 style="background-color: navy; color: white;">Hello World</h1>
  • Internal Styling - Animal Page

    Internal styling is a method of styling with CSS, and it is done in the head section of your HTML file.

    In this code snippet, the body of the page is styled to have a font of Arial and a background color of pink, and the h1 is styled to have a font size of 24 pixels and a font color of Rebecca purple

    <head>   <style>     body {       font-family: Arial;       background-color: pink;      }     h1 {       font-size: 24px;       color: rebeccaPurple;     }   </style> </head>
  • External Styling - All About You 2.0

    External styling is another method of styling with CSS, and just as the name suggests, the styling is done externally, or outside of the HTML file. Usually this styling is done in a CSS file, such as a file called style.css

    Styling separately from your HTML allows for better organization of your structure and content (HTML) and your styles (CSS). It more easily allows you to quickly update your styles and/or content when the HTML and CSS are separated.

    The example below shows CSS that styles various aspects of a paragraph tag and also centers an image. It looks almost the same as the internal styling example, BUT the style is located in the style.css file.

    /* This style is located in style.css */ p {   background-color: #efefef;   border-style: dotted;   border-color: black;   border-width: 3px;   border-radius: 10px; } img {   width: 50%;   margin-left: auto;   margin-right: auto; }
